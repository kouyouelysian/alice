The purpose of this tool is to build some logic circuits and have fun with them. For now, only the basic stuff like logic gates are available. You pick the device from the Devices folder in the project tree, and place them onto the simulator grid. Connect the inputs and outputs together to form circuits, and use sources and lights as input and output devices!

Command list (press keys)

  • W: route wire
  • H - highlight
  • Hover + Del, Backspace: delete
  • S: put source (basic signal generator)
  • L: put light (basic signal indicator)
